# Story 2.5.3 : Tests d'intégration workflow complet

**Status:** Draft  
**Epic:** Epic 2 - Système de Simulation et Prédiction

---

## Story

**As a** développeur/QA,  
**I want** tests d'intégration pour le workflow complet simulation → features → labels → ML,  
**so that** le système fonctionne de bout en bout sans erreurs.

---

## Acceptance Criteria

1. Tests d'intégration workflow complet : Simulation → Features → Labels → ML.
2. Tests avec données mockées et fixtures pré-calculées.
3. Validation end-to-end : Vérifier que toutes les étapes s'enchaînent correctement.
4. Tests de régression : Vérifier qu'une modification ne casse pas le workflow.
5. Documentation tests : README tests, commentaires, exemples.
6. Tests automatisés : Exécutables via pytest.

---

## Integration Verification

IV1: Workflow complet fonctionne de bout en bout (simulation → ML).  
IV2: Tests détectent régressions (modifications qui cassent le workflow).
